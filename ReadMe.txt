# backend

backend du Projet 6 

## Prérequis

Avant de commencer, assure-toi d'avoir installé les outils suivants sur ta machine :

- [Node.js](https://nodejs.org/) (version 14 ou supérieure recommandée)
- [npm](https://www.npmjs.com/) (automatiquement installé avec Node.js)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) ou une autre instance MongoDB

## Installation

1. Clone le repository 
2. Lance npm install, ou npm install --force si nécessaire
3. Mets le bon .env
